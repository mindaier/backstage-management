# rancho-admin-web
racnho-admin-web是基于vue、element-ui等开发的一套最小前端管理系统，其中功能包括登陆、管理员管理、角色管理<br>
后端项目源码：[https://github.com/rancho00/rancho-web](https://github.com/rancho00/rancho-web)

# 搭建步骤
    # 安装依赖
    npm install
    # 启动服务
    npm run dev
    # 构建生产环境
    npm run build:prod
